HTML Elements & Tags

What is an HTML Tag?

HTML tags tell the browser how to display content. Most tags have an opening tag and a closing tag. 

<h1>This is a Heading</h1>

<p>This is a paragraph.</p>

<a href="https://example.com">Visit Website</a>
