🔤 CSS Lesson 4 - Fonts & Text

Learn how to make text beautiful and easy to read.

Changing Font Size

h1{

font-size:40px;

}

Changing Font Family

body{

font-family:Arial,sans-serif;

}

Making Text Bold

p{

font-weight:bold;

}
