What is CSS?

CSS stands for Cascading Style Sheets. It controls the colors, fonts, spacing, layouts and animations of a webpage.

💡 Tip: HTML builds the structure while CSS makes it beautiful.

Example

h1{

color:blue;

font-size:40px;

}
