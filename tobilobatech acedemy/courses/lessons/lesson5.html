📊 Lesson 5 - HTML Tables

💎 Premium Lesson • Learn how to organize data into rows and columns.

📚 What is an HTML Table?

HTML tables are used to display information in rows and columns, making data easy to read.

💡 Tip: Tables are great for timetables, price lists, exam results and reports.

Example

<table border="1">

<tr>
<th>Name</th>
<th>Age</th>
</tr>

<tr>
<td>Tobi</td>
<td>16</td>
</tr>

</table>

Output

Name Age
Tobi 16
David 18

Explanation

  • <table> creates the table.
  • <tr> creates a row.
  • <th> creates a heading cell.
  • <td> creates a normal data cell.
⚠ Common Mistake: Don't forget to close every <tr>, <th> and <td> tag.
