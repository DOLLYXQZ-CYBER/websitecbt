Common Input Types

HTML provides different input types for collecting different kinds of information.

💡 Tip: Using the correct input type improves user experience and helps browsers validate data.

Example

<input type="text" placeholder="Full Name">

<input type="email" required>

<input type="password">

<input type="number" min="1" max="100">

<input type="date">

<input type="color">

Useful Attributes

Attribute Purpose
placeholder Shows a hint inside the input
required Makes the field compulsory
readonly User cannot edit the value
disabled Disables the input field
maxlength Limits the number of characters
min / max Sets the minimum and maximum values
⚠ Common Mistake: Don't use type="text" for everything. Use the correct input type for better validation and usability.
