What is Semantic HTML?

Semantic HTML uses meaningful tags that describe the purpose of each section of a webpage. Instead of using many <div> tags, semantic tags make your code cleaner and improve SEO.

💡 Tip: Semantic HTML helps search engines understand your website better.

Example

<header>

<nav>Navigation</nav>

</header>

<main>

<section>

Lesson Content

</section>

</main>

<footer>

Copyright

</footer>

Common Semantic Tags

Tag Purpose
<header> Top section of the page
<nav> Navigation links
<main> Main page content
<section> A section of content
<article> Independent content
<footer> Bottom of the page
⚠ Common Mistake: Don't build your whole website using only <div> elements. Use semantic tags whenever they fit the content.
